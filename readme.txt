= WooCommerce Custom Product Tab Manager =

== Description ==

You can create multiple custom tabs for products. Create one tab for all products or custom tabs for each product. Also you can create global custom tabs for all products or select specific categories. Use WordPress native default editor where you can edit html codes or visual edit also can upload media, add any type short-codes.

= Basic Features =

* Unlimited products custom tabs
* Global tabs management system
* Specific categories wise tabs
* Select for all products tabs
* WordPress default editor use for tab content
* Allow short-codes, html codes, visual mode, text mode

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==

= Q. Where do I go to add tabs to a product? =
A.  Go to the wp-admin area and product edit page, now you will find “Custom Tabs” in the bottom left corner of the WooCommerce default product tabs area. Click on “Custom Tabs” to reveal the custom tab manager.

= Q. Where will these tabs appear? =
A.  Tab will appear on the single product page frontend area.

= Q. Why does the WYSIWYG editor default to the ‘Visual’ tab? =
A.  Yes

= Q. Is this support global tab? =
A.  Yes, go to the Product Custom Tabs page under the WooCommerce product menu.

= Q. Can I add a custom tab for all products? =
A.  Yes, here can add custom tabs for all products globally.

= Q. Can I add a custom tab for specific categories? =
A.  Yes, here can add custom tabs for specific multiples categories.

= Q. Is it compatible with all WordPress themes? =
A.  Yes, all kinds of WordPress themes are compatible.
